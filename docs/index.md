# BTC Wine

## RWA Vault Infrastructure on Stellar

BTC Wine is a Real-World Asset (RWA) vault built on the Stellar blockchain using Soroban smart contracts. The vault tokenizes wine production assets and enables capital-efficient liquidity allocation.
